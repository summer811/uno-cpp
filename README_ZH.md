## 📌 项目简介
本项目为 C++ 实践课的期末大作业。我们团队在 C++ 的基础上，利用 EasyX 图形库从零实现了经典桌游 UNO（优诺牌）的核心逻辑与可视化界面。

⚠️ **特别说明**：为了尊重版权并打造独特的视觉风格，本项目**未使用任何官方原版游戏贴图**，游戏内的所有卡牌素材及美术资源均为团队成员**纯原创/自主绘制**。现已完全开源，欢迎学习与交流！

---

## ✨ 项目亮点与特性
* **完整规则还原**：支持基础牌（数字牌）及所有功能牌（跳过、反转、加 2、万能牌、王牌/加 4）。
* **纯原创美术**：告别千篇一律的官方素材，采用团队手绘/自制风格的卡牌视觉，界面独特。
* **面向对象架构**：采用清晰的面向对象设计（OOP），洗牌、发牌、出牌判定、回合管理逻辑解耦清晰。
* **流畅交互体验**：基于 EasyX 图形库实现动画与鼠标交互，告别枯燥的控制台命令行。

## 🚀 快速开始 / 编译运行指南

为了能够成功运行本项目，请按照以下步骤进行配置：

### 1. 前置准备：安装 EasyX 库
由于本项目基于 EasyX 驱动，编译前请确保您的 Visual Studio 2022 已配置 EasyX 环境：
1. 前往 EasyX 官网下载最新版安装包（或[点击此处直达官网](https://easyx.cn/)）。
2. 解压并运行安装程序，选择对应的 **Visual Studio 2022** 点击安装即可。

### 2. 克隆仓库
打开您的终端（Terminal）或 Git Bash，执行以下命令：
```bash
git clone https://github.com/summer811/uno-cpp.git
cd uno-cpp
```
### 3. 编译与运行
进入克隆好的项目根目录，找到 uno-cpp.sln（Visual Studio 解决方案文件）。
双击 uno-cpp.sln，系统会自动拉起 Visual Studio 2022。
在 Visual Studio 顶部工具栏中，确保解决方案配置选择的是 Debug 或 Release，平台选择的是 x64（或根据你们项目实际情况选择 x86）。
按下快捷键 Ctrl + F5（或点击顶部的“本地 Windows 调试器”）即可编译并启动游戏！
